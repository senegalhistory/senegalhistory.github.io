Mozilla Festival Data Expeditions

Welcome to the Data Expeditions session at #MozFest. We believe that gathering data is a great way to learn about how institutions work and how policies are implemented. In this workshop, we will

chart out the data sources and types available for a few topics. Our goal is to find out which information is available, how it connects and what we could learn from it.
